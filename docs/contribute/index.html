Contribution rules

Short checklist from the project contribution guide.

  1. Do not submit links that already exist in the list.
  2. Links must be working and active when submitted.
  3. Keep formatting consistent with list.md table structure.
  4. For locked links, include valid username/password or do not submit them.
  5. Give a contributor name/alias so credit can be assigned.

Blocked domains

b-cdn.net blooket.com

How to contribute

  1. Use GitHub for direct edits and pull requests.
  2. Use Google Form if you want a quick submission path.
  3. For GitHub, add rows in the existing table format under the right section.
  4. If you add a new section, include emoji + note table + links table.

Your entries sync to both list.md and the website data files.
